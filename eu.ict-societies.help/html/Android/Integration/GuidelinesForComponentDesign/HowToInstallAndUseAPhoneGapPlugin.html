How to Install/Use a PhoneGap Plugin¶

This article is a summarize, more focus on Societies needs, of the PhoneGap Wiki page: How to Install a PhoneGap Plugins for all platforms.

How to Install PhoneGap Plugins for Android¶

  • Copy/paste the JS file of the plugin somewhere in: assets/www/. A good practice is to paste it in the folder: assets/www/js/plugin
  • Ensure that your project can access the Java file of the plugin (e.g. through Maven dependency)
  • Edit (or create) the file res/xml/plugins.xml and add a <plugin> tag:
    									

1 <plugins>
2 ...
3 <plugin name= "YourPluginName"value="package.YourJavaPlugin"/>
4 ...
5 </plugins>
  • If needed run "mvn site" on the project of the plugin to easily show the Javascript API of this plugin
