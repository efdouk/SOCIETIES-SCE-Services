Android Communications Testing¶

AVD configuration¶

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/AndroidConfiguration

$ sh startTestAVD.sh <AVD name>

when the AVD is ready

$ sh initTestAVD.sh

Windows developers should consult Android Communications for more details on running these scripts

Compilation, deployment and running of testers¶

It is important that all OSGi and Android projects are compiled to ensure that components are up to date.

Remove all current artifacts from your local Maven repository¶

$ rm -rf $HOME/.m2/repository/org/societies

Compilation of Societies components¶

Compile the OSGi and Android components in the following order having set your current directory to SOCIETIES-Platform/builder:

$ mvn -fae clean install -f spring-osgi-projects.xml

$ mvn -fae clean install -f android-projects.xml

Android Agent Tester¶

This tester tests the remote invocation Android XMPP Communications component

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/AndroidAgent and compile

$ mvn clean install android:deploy

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/AndroidAgentTester and compile

$ mvn clean install android:deploy android:run

Examining the Android logging (adb logcat from command line or use the DDMS perspective in Eclipse) should display stanzas being sent and received

Android Pubsub Tester (Ignore until further notice)¶

This tester tests the Pubsub Android XMPP Communications component

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/AndroidPubsub and compile

$ mvn clean install android:deploy

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/AndroidPubsubTester and compile

$ mvn clean install android:deploy android:run

Examining the Android logging (adb logcat from command line or use the DDMS perspective in Eclipse) should display events being subscribed to and events being received

Applications¶

If the results from the Android Communications testers prove satisfactory, then compile and run the Android Societies Client app:

Set your current directory to SOCIETIES-Platform/platform-infrastructure/client-frwk/MasterGUI and compile

$ mvn clean install android:deploy android:run

Again, refer to the Android logging for confirmation of communications being sent and received.
