SOCIETIES Platform Configuration Properties

For any custom configuration, the folder VIRGO_HOME\config has to be used. To allow certain bundles that use a database to work correctly, a properties file named org.societies.platform.properties is required in the $VIRGO_HOME/config directory (N.B. The VIRGO_HOME environment variable must be created to allow this properties to work).

Its contents for Windows are:

jdbc.driverClassName=com.mysql.jdbc.Driver
jdbc.url=jdbc\:mysql\://127.0.0.1\:3306/societies1
jdbc.password=root
jdbc.username=root
hibernate.dialect=org.hibernate.dialect.MySQLDialect
hibernate.dialect=org.hibernate.dialect.MySQLDialect
hbm2ddl.auto=create
show_sql=true
current_session_context_class=thread

Whilst for Linux there are some minor changes. These are the entries that have to be altered:

jdbc.url=jdbc:derby:${VIRGO_HOME}/Databases/serviceregdb;create=true
hibernate.connection.url=jdbc:mysql:target/database/message;create=true

all other entries remain untouched.
