More Features

To be able to track only IDevice in which it is intersted, a 3P service has to use LDAP filters when tracking a service using ServiceTraker.
To do so, first of all import the two interfaces org.osgi.framework.Constants and org.osgi.framework.Filter
1. Create a String LDAP Filter including the OBJECTCLASS property that must be equal to IDevice.class.getName then and to the filter any property defined in the DeviceMgmtConstants class. You can filter by device type, device name, device id, device node id... and so on (see DeviceMgmtConstants class for more information).
2. Then use the BundleContext class to create a Filter object
3. Finaly use the Filter object created when you instantiate your ServiceTracker
