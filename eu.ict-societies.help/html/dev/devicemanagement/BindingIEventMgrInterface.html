Binding IEventMgr interface

The only OSGi service reference to be added in the bundle-context-osgi.xml file associated with the 3rd party service is:

<osgi:reference id="EventManager" interface="org.societies.api.osgi.event.IEventMgr"/>

Finally, the following bundles should be added in your Virgo plan file (together with other SOCIETIES platform bundles such as the Communication Manager, not listed here as they are mandatory):

<!-- Device Management bundles. --> <artifact type="bundle" name="org.societies.css.devicemgmt.DeviceCommsMgr"/> <artifact type="bundle" name="org.societies.css.devicemgmt.devicemanager"/>
